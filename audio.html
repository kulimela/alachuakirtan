Instructions for downloading audio tracks:

1. Click with the right mouse button on any of the audio links below.
2. Select "Save Link As..." (in Safari select "Download Linked File As...")
3. Choose where to save the MP3 file on your computer.
4. Save the file.
5. Repeat for all tracks you want to download.
